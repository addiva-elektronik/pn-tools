# pn-tools

A command-line tool for Profinet-related tasks

Supported commands:
 - **discovery**: Discovers Profinet devices on the network
 - **flashled**: Sends a "flash leds" request to a Profinet device

## Compiling

    sudo apt install build-essential
    make
